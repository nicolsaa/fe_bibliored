# Project Brief

BiblioRed Android-Kotlin memory bank

- Purpose: Provide a consolidated technical reference for the Android-Kotlin artifact of the BiblioRed project, facilitating maintenance, onboarding, and future evolutions.

- Scope: Domain models, data and network layers, DTOs/adapters, UI layers, ViewModels, session persistence, and key business flows.

- Audience: Current and future developers, QA, and product team.
